# Parking Chatbot RAG System

## Overview
An intelligent chatbot for parking info and reservations, using RAG architecture, LangChain, and Milvus.

## Setup
1. Install requirements: 'pip install -r requirements.txt'
2. Start Milvus server.
3. Run app: `python app.py`

## Features
- Info retrieval (prices, hours, location)
- Reservation flow
- Sensitive data guard rails
- Performance evaluation

## Tests
Run: `pytest tests/`
